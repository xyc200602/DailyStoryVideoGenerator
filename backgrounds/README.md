# 背景图片说明

## 支持的背景图片

请将背景图片放置在此目录下，支持的格式：
- JPG/JPEG
- PNG
- BMP

## 图片命名建议

按照场景类型命名图片，以便程序自动选择：
- `urban_night.jpg` - 都市夜景
- `fantasy_world.jpg` - 玄幻世界
- `ancient_china.jpg` - 古代中国
- `modern_office.jpg` - 现代办公室
- `school_campus.jpg` - 校园
- `battle_arena.jpg` - 战斗场景
- `cultivation_mount.jpg` - 修仙山
- `future_city.jpg` - 未来城市

## 图片尺寸建议

- 推荐分辨率：1920x1080 或更高
- 支持的宽高比：16:9
- 最小尺寸：1280x720

## 注意事项

1. 请确保使用的图片拥有合法版权
2. 建议使用色调较暗的背景，以便文字显示更清晰
3. 可以准备多张同一类型的图片，程序会随机选择
